##ATM Management System

##Project Description 
   ATM Management System is a Java Swing based desktop application that simulates basic ATM functionlities.
   The system allows users to securely log in and perform banking operations similar to a real ATM.

##Features
-User Login with Username and PIN
-Deposit Amount
-Withdraw Amount
-Balance Enquiry
-Fund Transfer
-Transaction History 
-Exit with Thank You Message

##Technologies Used
-Java
-Java Swing(GUI)
-JDBC
-MySQL
-Eclipse IDE

##Project Structure
-src/ -Java source files
-bin/ -Complies class files
-db-Database connection classes
-ui-User interface frames
-service -Business logic

##How to Run the Project
1.Open the project in Eclipse IDE
2. Congigure MySQL database connection
3. Run the main class (BankApp.java)
4. Login using valid username and PIN
